Event Flyer

From my "Intro to Graphic Media" class in high school

Project Overview

For this assignment, we were told to use our imagination to make up an event and create an appealing flyer advertising this event.

Tools Used

Adobe Photoshop ● Adobe Illustrator

I chose to create a flyer advertising the grand opening of a fictional amusement park. I used Adobe Photoshop and Adobe Illustrator to create my flyer and add eye catching imagery and text which would make it quite effective if it was for a real event.

Blender Tree

From experimenting with Blender

Project Overview

This project was just me experimenting with Blender for the first time because I wanted to learn how to use the program. I decided to make a 3D birch tree, something cartoonish which looks like it could be in video game.

Tools Used

Blender

These four images are of the same tree, just photographed from different angles with different light sources. After making an autumnal birch tree using simple colors, I added textures to make it look more realistic.
